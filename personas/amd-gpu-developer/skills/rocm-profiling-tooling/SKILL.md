# ROCm Profiling Tooling

Use this skill when collecting traces and counters with ROCm profiling tools.

## Focus

- `rocprof`/`rocprofv2` usage and trace capture
- Counter collection and kernel attribution
- Correlating timeline events with kernel source changes

## Checklist

1. Run profiling with stable input and warm-up behavior.
2. Capture both timeline and selected hardware counters.
3. Attribute hotspots to specific kernels and call sites.
4. Summarize findings as actionable tuning tasks.
