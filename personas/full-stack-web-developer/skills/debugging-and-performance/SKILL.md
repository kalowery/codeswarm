# Debugging and Performance

Use this skill for diagnosing production issues and performance regressions.

## Focus

- Reproduction strategy and hypothesis-driven debugging
- Request-path latency and throughput bottleneck analysis
- Frontend/backend/database correlation for end-to-end performance

## Workflow

1. Reproduce issue with minimal, deterministic steps.
2. Isolate bottleneck using traces/logs/profiles.
3. Implement targeted fix and verify with benchmarks.
4. Add guardrails to prevent recurrence.
