# Deployment and Operations

Use this skill for packaging, releasing, and operating web services.

## Focus

- Release safety (health checks, rollback paths)
- Environment configuration and secrets handling
- Monitoring, alerting, and incident response readiness

## Workflow

1. Define deployment plan and rollback criteria.
2. Validate configuration across environments.
3. Add/update dashboards and alerts for changed behavior.
4. Capture post-deploy verification steps.
