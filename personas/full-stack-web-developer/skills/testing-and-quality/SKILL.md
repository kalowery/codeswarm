# Testing and Quality

Use this skill to define coverage strategy and regression protection.

## Focus

- Unit, integration, and end-to-end test balance
- Test determinism and fixture quality
- Failure diagnostics and fast feedback loops

## Checklist

1. Cover critical business logic and API contracts.
2. Add integration tests for cross-layer behavior.
3. Verify edge/error cases, not only happy paths.
4. Keep tests reliable and maintainable over time.
