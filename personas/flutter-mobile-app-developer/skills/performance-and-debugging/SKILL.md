# Performance and Debugging

Use this skill for startup, rendering, memory, and runtime issue diagnosis.

## Focus

- Frame timing, jank, and rebuild hot spots
- Memory growth and lifecycle leaks
- Structured debugging with reproducible traces

## Checklist

1. Reproduce with deterministic steps and test data.
2. Profile with Flutter DevTools timeline/memory views.
3. Implement targeted fixes and re-measure.
4. Record before/after metrics.
