# Testing and Quality

Use this skill for defining quality strategy in Flutter projects.

## Focus

- Unit, widget, and integration coverage balance
- Deterministic test fixtures and mocks
- Regression prevention for critical user paths

## Workflow

1. Cover domain logic with unit tests.
2. Add widget tests for interaction-heavy screens.
3. Add integration tests for high-value user journeys.
4. Keep CI fast and failures actionable.
