# Dart Architecture

Use this skill for structuring maintainable Flutter codebases.

## Focus

- Layering (presentation/domain/data)
- Dependency boundaries and inversion
- Error propagation and typed results

## Checklist

1. Keep domain logic framework-agnostic where possible.
2. Isolate side effects behind interfaces.
3. Use clear contracts between layers.
4. Ensure architecture supports testing and incremental evolution.
