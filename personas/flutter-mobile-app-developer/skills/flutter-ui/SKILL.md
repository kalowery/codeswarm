# Flutter UI

Use this skill for building Flutter screens and reusable widgets.

## Focus

- Widget composition and layout responsiveness
- Accessibility semantics and interaction ergonomics
- Theming, typography, and design consistency

## Workflow

1. Define UI states (loading, empty, error, success).
2. Build reusable widgets with clear public APIs.
3. Verify behavior across form factors and orientations.
4. Add widget tests for critical interactions.
