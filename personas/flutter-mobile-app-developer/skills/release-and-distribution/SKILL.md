# Release and Distribution

Use this skill for preparing and shipping app releases.

## Focus

- Build variants, signing, and environment config
- Store submission readiness and metadata
- Crash monitoring and rollout safeguards

## Workflow

1. Validate release build configuration and signing.
2. Run smoke and regression checks on release artifacts.
3. Prepare staged rollout and monitoring plan.
4. Define rollback/mitigation actions for release risk.
